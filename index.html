Apresentação do curso

Olá, seja bem-vindo(a) ao curso de Introdução à análise de dados para pesquisa no SUS.

Este é um curso autoinstrucional destinado a profissionais do Sistema Único de Saúde (SUS), estudantes de graduação em saúde e a todos que desejam se aprofundar no tema e tem por finalidade apresentar conceitos básicos sobre modelos estatísticos no contexto da saúde pública, assim como a utilização de software livre e boas práticas de visualização de dados para resolver problemas de forma sistemática em saúde.

Ao longo do curso você terá a oportunidade de colocar em pratica o seu aprendizado, de forma opcional. Para isso, disponibilizaremos um material complementar hospedados no GitHub para que você consiga executar os scripts e ao final do curso, você terá acesso ao gabarito dessas atividades práticas para a conferência de resultados.

E por que aprender análise de dados com apoio de softwares?

Vivemos na era da informação. Todos os dias, o SUS gera uma imensa quantidade de dados, como registros de atendimentos, cadastros de usuários, estoques de medicamentos, dados ambientais e de vigilância em saúde, entre outros. Essa abundância de dados representa uma oportunidade valiosa para aprimorar o planejamento, a gestão e a tomada de decisões no sistema de saúde pública.

No entanto, analisar dados manualmente com papel, caneta ou mesmo planilhas simples não é mais viável quando lidamos com grandes volumes de informações. É aqui que entram os softwares de análise de dados.

Esses softwares permitem:

  • Organizar e limpar grandes bases de dados.
  • Reproduzir análises com scripts reutilizáveis, garantindo mais transparência e agilidade.
  • Aplicar modelos estatísticos de forma segura.
  • Produzir gráficos e visualizações que facilitam a comunicação dos resultados.

Porém, para usar essas ferramentas com autonomia é preciso compreender a lógica de programação. Programar não é “falar com o computador” em linguagem complicada — é, na verdade, uma forma de organizar o raciocínio e automatizar tarefas repetitivas. Com programação, você pode:

  • Repetir análises com um clique.
  • Compartilhar scripts com colegas para facilitar o trabalho em equipe.
  • Modificar rotinas/análises sem ter que partir do zero.
  • Criar relatórios personalizados e atualizáveis.

Neste curso você será introduzido ao uso de linguagens e ferramentas que tornam tudo isso possível. A lógica de programação servirá de base para os módulos de análise estatística e visualização de dados.

Aprender programação e análise de dados não é um luxo, é uma necessidade para quem quer usar os dados do SUS de forma estratégica e inteligente.

Organize sua agenda de estudos e aproveite os conteúdos no seu ritmo. Reflita, pratique, explore os materiais complementares e, principalmente, conecte o aprendizado com a sua realidade profissional.

Bons estudos!
